# Menox-s-macroboard

This is my personal Macroboard controlled by a XIAO R2040. I redid it multiple times and this is the final version I am quite proud of. thanks to @Simon who helped me alot and thanks to whom I was able to improve my skills drasticly.
It got 9 keys, each with RGB-LEDs, a rotary encoder and a 0.91 OLED. This macroboard will be used to improve my workflow and just add more features to my setup.
